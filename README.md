# Vectify
Let's show your spotify information on the Vector's screen!

## Install
- clone this repo
- cd /path/to/this/repo
- `pip install .`
- setup spotify app + info

## Run Example
- `cd /path/to/this/repo`
- `python examples/main.py`

## How to Use
See [examples](https://github.com/ryosakagami/vectify/tree/master/examples)!
